# DPS

Удобное отображение Docker-контейнеров.

## Установка

linux

```bash
sudo curl -L# -o /usr/bin/dps https://github.com/marvell/dps/releases/download/1.0/dps_linux_amd64
sudo chmod +x /usr/bin/dps
```

osx

```bash
sudo curl -L# -o /usr/bin/dps https://github.com/marvell/dps/releases/download/1.0/dps_linux_amd64
sudo chmod +x /usr/bin/dps
```
